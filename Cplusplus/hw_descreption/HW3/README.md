# 1.圖形製作v3
輸入一正整數N，

請輸出以下圖形
```
輸入範例:
5

輸出範例:
1
212
32123
4321234
543212345
----------------------------------------------------------------
輸入範例:
2

輸出範例:
1
212
----------------------------------------------------------------
輸入範例:
3

輸出範例:
1
212
32123
```

再輸入一整數來輸出圖形範圍
```

    5

  33333
   4444
    555
     66
      7

 *
```
